# Change Log

All notable changes to the "roast-my-codebase" extension will be documented in this file.

## [0.1.0] - 2026-06-26

### Added
- Initial release of VS Code extension
- Workspace scanning with CLI integration
- Real-time health score in status bar
- Inline diagnostics for code quality issues
- Tree view panels for findings and statistics
- Interactive HTML report viewer
- Badge generation command
- Configurable settings for scan behavior
- Cache management for scan results

### Features
- **Commands**: Scan Workspace, Scan File, Show Report, Generate Badge, Clear Cache
- **Views**: Findings organized by category, Statistics panel
- **Status Bar**: Color-coded health score with click-to-report
- **Diagnostics**: Inline warnings, errors, and info markers
- **Configuration**: Full settings support for enabling/disabling features
